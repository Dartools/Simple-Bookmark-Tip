# Simple-Bookmark-Tip
一个轻量级的 Ctrl+D / Cmd+D 收藏引导提示，支持自定义位置、颜色、设备显示限制及复现时间。
适合wordpress网站中使用。

**使用方法：**

在你的服务器后台 wp-content/plugins 创建一个新的插件文件夹，如【simple-bookmark-tip】，然后将 simple-bookmark-tip.php 放入即可。

WP后台 - 插件 - 已安装插件 启用【 simple-bookmark-tip】

WP后台 - 设置 - 收藏引导设置，可进行设置。
